1.Citește un număr n, apoi un vector de n elemente întregi și afișează elementele pe o linie, separate prin spațiu.
2.Citește un vector de n numere întregi și afișează suma tuturor elementelor.
3.Citește un vector de n elemente și afișează doar valorile pare.
4.Citește un vector de n numere întregi și afișează valoarea maximă.
5.Citește un vector de n elemente și afișează elementele de la ultimul la primul.
6.Citește un vector de n elemente și afișează câte dintre ele sunt strict mai mari decât 0.
7.Citește un vector de n numere reale și afișează media aritmetică a elementelor.
8.Citește un vector de n elemente și afișează "DA" dacă toate elementele sunt egale, altfel "NU".
9.Citește un vector de n numere întregi și afișează prima poziție pe care se află valoarea maximă (indexul începe de la 0).
10.Citește un vector de n elemente și afișează doar valorile aflate pe poziții pare (0, 2, 4, ...).
11.Citește n și un vector, apoi o valoare x. Afișează vectorul fără aparițiile lui x.
12.Afișează "DA" dacă elementele sunt în ordine crescătoare, altfel "NU".
13.Citește un vector și afișează-l după ce ai schimbat primul element cu ultimul.
14.Citește un vector și un număr x, apoi inserează x pe poziția k.
15.Citește un vector de n elemente și determină lungimea celei mai lungi secvențe de elemente egale consecutive.
16.Citește un vector și determină lungimea celei mai lungi secvențe de elemente aflate în ordine strict crescătoare.
17.Vectorul dublat: creează un nou vector în care fiecare element apare de două ori consecutiv. (Ex: [3, 7, 1] → [3, 3, 7, 7, 1, 1].)
18.Citește un vector și o valoare S. Verifică dacă există 3 elemente consecutive a căror sumă > S.
19.Citește doi vectori ordonați crescător și construiește un al treilea vector care conține toate elementele celor doi, tot în ordine crescătoare.
20.Citește un vector de n elemente și verifică dacă este o permutare a mulțimii {1, 2, ..., n}.
21.Citește un vector cu elemente între 1 și 100. Afișează de câte ori apare fiecare valoare .
