1.Citește un număr n, apoi un vector de n elemente întregi și afișează elementele pe o linie, separate prin spațiu.
2.Citește o matrice 3x3 de numere întregi și afișeaz-o.
3.Citește o matrice n x m și afișează toate elementele de pe prima linie.
4.Citește o matrice n x m și afișează toate elementele de pe ultima coloană.
5.Citește o matrice n x m și afișează suma tuturor elementelor.
6.Citește o matrice n x m și afișează toate elementele pare.
7.Citește o matrice n x m și afișează elementele de pe linia k (citită de la tastatură).
8.Citește o matrice n x m și afișează elementele de pe coloana k (citită de la tastatură).
9.Citește o matrice 3x3 și afișează elementele în ordine inversă, adică de la ultimul element la primul.
10.Citește o matrice pătratică n x n și afișează suma elementelor de pe toate cele 4 laturi (marginea matricei).
11.Citește o matrice n x m și găsește elementul maxim și poziția lui (linia și coloana).
12.Citește o matrice n x m și numără câte elemente sunt negative.
13.Citește două matrici n x m și afișează matricea rezultată prin adunarea celor două.
14.Citește o matrice pătratică n x n și afișează matricea transpusă.
15.Citește o matrice n x m și afișează toate elementele care sunt mai mari decât media tuturor elementelor matricei.
16.Citește o matrice pătratică n x n și verifică dacă toate elementele de pe diagonala principală sunt egale.
17.Citește o matrice pătratică n x n și inversează elementele de pe diagonala principală (prima cu ultima, a doua cu penultima, etc.).
18.Citește o matrice pătratică n x n și afișează elementele de pe diagonala principală.
19.Citește o matrice pătratică n x n și afișează elementele de pe diagonala secundară.
20.Citește o matrice pătratică n x n și calculează suma elementelor de pe diagonala principală.
21.Citește o matrice pătratică n x n și calculează suma elementelor de pe diagonala secundară.
22.Citește o matrice pătratică n x n și afișează diferența dintre suma elementelor de pe diagonala principală și suma elementelor de pe diagonala secundară.
23.Citește o matrice pătratică n x n și verifică dacă matricea este simetrică față de diagonala principală.
24.Citește o matrice pătratică n x n și afișează suma elementelor aflate pe ambele diagonale fără să dublezi elementul comun (dacă n este impar).
25.Citește o matrice pătratică n x n și construiește o matrice nouă în care toate elementele în afara celor de pe diagonalele principală și secundară sunt zero.
