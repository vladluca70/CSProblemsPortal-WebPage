1. Declară o variabilă x și atribuie-i valoarea 10. Afișează valoarea lui x.
2. Declară o variabilă y, atribuie-i valoarea 5. Apoi schimbă valoarea lui y în 8. Afișează valoarea finală.
3. Declară a = 3 și b = 7. Atribuie valoarea lui b în a, apoi afișează a și b.
4. Declară două variabile m = 4 și n = 6. Creează o a treia variabilă suma și atribuie-i rezultatul m + n. Afișează suma.
5. Declară rezultat = 3 + 2 * 4. Afișează rezultat.
6. Declară z = 10. Apoi, scrie z = z + 1. Afișează noua valoare a lui z.
7. Declară z = 10. Apoi, scrie z = z ++. Afișează noua valoare a lui z.
8. Declară două variabile a = 7 și b = 3. Schimbă valorile lor folosind o variabilă intermediară temp. Afișează valorile după schimb.
9. Declară x = 4, y = 8, z = 6. Calculează media aritmetică (folosind float) și atribuie rezultatul unei variabile media. Afișează rezultatul.
10. Declară int x = 3;. Fă x = x + 1; și apoi x = x * 2;. Afișează rezultatul final.
11. Declară o variabilă float celsius = 25.0; Atribuie lui fahrenheit formula: fahrenheit = celsius * 9 / 5 + 32; Afișează rezultatul.
12. Ai a = 1, b = 2, c = 3. Fă o rotație a valorilor: a → b, b → c, c → a, folosind o singură variabilă auxiliară. Afișează rezultatul final.
13. Declară int a, b, c; Atribuie toate trei în aceeași linie: a = b = c = 5; Afișează a, b, c.
14. Declară int a = 10, b = 20; Fă swap fără variabilă auxiliară (folosește a = a + b; b = a - b; a = a - b;) Afișează valorile finale.
15. Declară int x = 10; Fă x += 5; Afișează x.
16. Declară float a=3.5 și int b. Atribuie a lui b (adica b=a) și afișează b. Ce observi?
